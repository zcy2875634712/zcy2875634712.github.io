# 线性表

线性表*（linear list）*是数据结构的一种，一个线性表是n个具有相同特性的数据元素的有限序列。

我们说“线性”和“非线性”，只在逻辑层次上讨论，而不考虑存储层次，所以双向链表和循环链表依旧是线性表。

## 栈
### 什么是栈
- 栈是具有一定操作约束的线性表
- 栈只在一端(栈顶)做插入和删除
- 栈是一种后进先出( LIFO)的线性表

栈这种数据结构有点类似一叠碗，我们要用时，只能从最上面拿(栈的出栈)。当我们要放回去时，我们也只能放在最上面(栈的入栈)。
### 顺序栈

```c
typedef struct {
    Elemtype data[MaxSize];
    int top;
} SqStack;
SqStack S;
```

top的取值

- 当栈不为空时，top为栈顶元素的下标
- 栈空：`S.top == -1`
- 栈满：`S.top == MaxSize - 1`
- 栈长：`S.top + 1`

栈的操作

- 初始化

  ```c
  void InitStack() {
      S.top == -1;
  }
  ```

  

- 入栈

  ```c
  bool push(Elemtype x) {
      if (S.top == MaxSize - 1)
          return false;
      S.data[++S.top] = x;
      return true;
  }
  ```

- 出栈

  ```c
  bool pop(Elemtype *x) {
      if (S.top == -1)
          return false;
      *x = S.data[S.top--];
      return true;
  }
  ```

  

### 链栈

链栈实现：带头结点和不带头结点

#### 不带头结点

```c
typedef struct stack *Pstack;
struct stack {
    int data;
    Pstack next;
};

Pstack top = NULL;

void push(int data){
    Pstack in;
    in = (Pstack) malloc(sizeof(struct stack));
    in -> next = top;
    in -> data = data;
    top = in;
}

bool pop(int *data){
    if (top == NULL)
        return false;
    Pstack out = top;
    top = top -> next;
    *data = out -> data;
    free(out);
    return true;
}

void freeStack() {
    Pstack del;
    while (top != NULL) {
        del = top;
        top = top -> next;
        free(del);
    }
}
```



## 队列

### 队列定义

队列*（queue）*，是一种先进先出*（FIFO, First-In-First-Out）*的线性表。在具体应用中通常用链表或者数组来实现。队列只允许在后端*（称为rear）*进行插入操作，在前端*（称为front）*进行删除操作。

### 循环队列

```c
typedef struct {
    Elemtype data[MaxSize];
    int rear;
    int front;
} Queue;
Queue Q;	
```

- 初始时：`Q.front = Q.rear = 0`
- 出队时：`Q.front = (Q.front + 1) % MaxSize`
- 入队时：`Q.rear = (Q.rear + 1) % MaxSize`
- 队列长度：`(Q.rear + MaxSize - Q.front) % MaxSize`

![](https://gitee.com/zcy2875634712/blog-image/raw/master/img/queue.png)

 - 队列满：`(Q.rear + 1) % MaxSize == Q.front`
 - 队列空：`Q.front == Q.rear`
